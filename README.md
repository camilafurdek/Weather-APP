# Weather APP

This is a simple weather application built with **Python**, using **Tkinter**, **ttkbootstrap**, **Pillow**, and the **OpenWeatherMap API**.
It allows you to search for the weather of any city by typing its name and pressing Enter or clicking the Search button.

---

## Features

- Search weather by city name
- Trigger search with Enter key or Search button
- Displays:
  - City and country
  - Temperature in Â°C
  - Weather description
  - Weather icon
---

## Technologies Used

- Python 3
- Tkinter
- ttkbootstrap
- Requests
- Pillow (PIL)
- OpenWeatherMap API

## How It Works

1. Enter a city name.
2. Press Enter or click Search.
3. The app displays:
    - Temperature in Celsius
    - Weather description
    - Weather icon (clouds, sun, rain, etc.)
